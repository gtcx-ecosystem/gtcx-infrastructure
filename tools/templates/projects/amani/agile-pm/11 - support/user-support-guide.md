## 1. Getting Started with AMANI

### What is AMANI?
AMANI (meaning "Peace" in Swahili) is your AI-powered guide for land services. Available 24/7 in 50+ languages through WhatsApp, SMS, USSD, or web chat.

### How to Access AMANI

#### WhatsApp
```
1. Save number: +254-xxx-xxx-xxx
2. Send "Hello" or "Habari"
3. Choose your language
4. Start your journey
```

#### USSD (Feature Phone)
```
Dial: *384#
Follow menu prompts:
1. Language selection
2. Service selection
3. Get guidance
```

#### SMS
```
Send to: 38400
Format: HELP <your question>
Example: HELP land registration
```

#### Web Chat
```
Visit: https://amani.gtcx.global
No registration required
Works on any device
```

## 5. Troubleshooting

### Common Issues

| Problem | Solution |

## 7. FAQ

**Q: Is AMANI free?**
A: Basic services are free. Premium features (unlimited queries, priority support) require subscription.

**Q: Can AMANI complete transactions for me?**
A: AMANI guides you but doesn't process payments or submit applications. You maintain control.

**Q: What if I don't have a smartphone?**
A: Use USSD (*384#) on any phone, or SMS to 38400.

**Q: Can AMANI help with court cases?**
A: AMANI provides general guidance but isn't a lawyer. For legal advice, consult a qualified attorney.

**Q: How accurate is AMANI?**
A: 99.5% accurate for standard queries. Complex cases are reviewed by human experts.

**Q: Can I talk to a human?**
A: Type "Human help" to connect with support agents (business hours).
