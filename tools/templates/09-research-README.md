## Research Principles

### P26: RESEARCHED

> Evidence-based decisions. Academic rigor. Validated assumptions.

Research informs every major architectural decision. We don't build on assumptions — we build on evidence.

## Contributing Research

1. **Propose**: Create an issue describing the research question
2. **Research**: Conduct investigation with documented methodology
3. **Document**: Write up findings with proper citations
4. **Review**: Peer review before publication
5. **Publish**: Internal first, then external if appropriate
