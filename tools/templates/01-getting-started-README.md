## Prerequisites

### Required
- Node.js 20+ (see `.nvmrc`)
- pnpm 8+
- Git

### Recommended
- Rust toolchain (for crypto development)
- Docker (for local services)
- VS Code with recommended extensions


## First Steps

1. **Clone and Install**
   ```bash
   git clone https://github.com/gtcx/gtcx.git
   cd gtcx
   pnpm install
   ```

2. **Read the Principles**
   - [30 Engineering Principles](../05-engineering/PRINCIPLES.md)
   - [Code Standards](../05-engineering/CODE-STANDARDS.md)

3. **Understand the Architecture**
   - [Architecture Overview](../02-architecture/README.md)
   - [Protocol Spec](../../gtcx-protocol-docs/spec/02-architecture.md)

4. **Pick Your Path**
   - Protocol Development → [03-protocols/](../03-protocols/)
   - Platform Development → [04-platforms/](../04-platforms/)
   - Infrastructure → [06-deployment/](../06-deployment/)
