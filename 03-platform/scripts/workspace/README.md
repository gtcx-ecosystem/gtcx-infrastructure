# Workspace PM scripts

BaselineOS PM v3 sync helpers wired from root `package.json` (`workspace:check`, `pm:sync`, `pm:status`).

| Script          | Purpose                             |
| --------------- | ----------------------------------- |
| `check.mjs`     | Validate workspace domain alignment |
| `pm-sync.mjs`   | Sync PM artifacts to core           |
| `pm-status.mjs` | Print PM sync status                |

Operational workspace tree: [`workspace/`](../../workspace/).
