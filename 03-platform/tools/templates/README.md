# `03-platform/tools/templates/` — Documentation scaffolds

Markdown templates for bootstrapping GTCX repo docs (not a workspace package — ADR-019 exemption).

| Path                                                     | Purpose                                        |
| -------------------------------------------------------- | ---------------------------------------------- |
| `01-getting-started-README.md` … `09-research-README.md` | Section stubs for new repos                    |
| `docs-README.md`                                         | Root docs index pattern                        |
| `PRINCIPLES.md`                                          | Shared doc principles                          |
| `projects/*/`                                            | Per-project template packs (amani, kora, maba) |

Copy and customize when standing up a new package or docs tree. Do not import at runtime.
