## Governance

- **Owner:** CISO
- **Approval:** Board Security Committee
- **Review cycle:** Annual (or upon material change)
- **Distribution:** All GTCX personnel and contractors
- **Exceptions:** Must be documented, risk-assessed, and approved by CISO + policy owner

## Version Control

All policies are version-controlled in this repository. Changes require a pull request with CISO approval before merge.
