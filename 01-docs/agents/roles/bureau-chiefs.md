# Bureau Chiefs

---

## Role Overview

Bureau Chiefs lead [Organization Name]'s coverage of specific geographic regions or thematic areas. They combine deep local expertise with editorial judgment to ensure our intelligence products reflect ground truth.

---

## Responsibilities

### Editorial Leadership

| Responsibility         | Description                                 |
| ---------------------- | ------------------------------------------- |
| **Coverage strategy**  | Define priorities for their region/beat     |
| **Source development** | Build and maintain source networks          |
| **Quality assurance**  | Ensure accuracy of regional content         |
| **Breaking news**      | First responder for regional developments   |
| **Context provision**  | Provide background for AI-generated content |

### Team Management

| Responsibility          | Description                                           |
| ----------------------- | ----------------------------------------------------- |
| **Contributor network** | Recruit and manage regional contributors              |
| **Training**            | Onboard contributors to [Organization Name] standards |
| **Performance**         | Monitor contributor quality and output                |
| **Relationships**       | Maintain industry and government contacts             |

### Content Production

| Responsibility                   | Description                             |
| -------------------------------- | --------------------------------------- |
| **Field dispatches**             | First-person reporting from the ground  |
| **Source verification**          | Validate information before publication |
| **Expert commentary**            | Provide analysis for sector reports     |
| **[Intelligence product] input** | Flag regulatory changes in real-time    |
