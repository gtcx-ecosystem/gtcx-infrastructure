## Overview

Contributors are [Organization Name]'s eyes and ears across [target markets]. They provide the ground-truth intelligence that no algorithm can replicate — local knowledge, source relationships, and contextual understanding.

---

## Contributor Types

### Field Correspondents

**Role**: On-the-ground reporting from [relevant locations and trade corridors].

| Responsibility     | Output                          |
| ------------------ | ------------------------------- |
| Field dispatches   | First-person reports from sites |
| Price surveys      | Local market price data         |
| Source cultivation | Relationship building           |
| Photo/video        | Visual documentation            |

**Profile**: Based in or near [target regions], trusted by local communities, mobile-first workflow.

### Industry Analysts

**Role**: Expert analysis and commentary on sector developments.

| Responsibility       | Output                         |
| -------------------- | ------------------------------ |
| Sector analysis      | Deep-dive reports              |
| Expert commentary    | Quotes and context for stories |
| Data interpretation  | Making sense of statistics     |
| Trend identification | Pattern recognition            |

**Profile**: Industry experience in [relevant sectors], analytical skills, written communication.

### Regulatory Monitors

**Role**: Track and interpret regulatory changes across jurisdictions.

| Responsibility      | Output                          |
| ------------------- | ------------------------------- |
| Regulatory alerts   | First notification of changes   |
| Interpretation      | What new rules mean in practice |
| Government liaison  | Official source relationships   |
| Compliance guidance | Practical implications          |

**Profile**: Legal/regulatory background, government relationships, local language skills.

### Data Contributors

**Role**: Provide structured data for indices and intelligence products.

| Responsibility   | Output                        |
| ---------------- | ----------------------------- |
| Price data       | Daily/weekly price points     |
| Volume data      | Production and trade volumes  |
| Survey responses | Structured data collection    |
| Verification     | Cross-checking published data |

**Profile**: Access to market data, reliability, consistency.

## Quality Standards

### Accuracy Requirements

| Standard                  | Requirement                    |
| ------------------------- | ------------------------------ |
| Factual accuracy          | 100% verifiable claims         |
| Source attribution        | Named or described sources     |
| Date/location specificity | Precise when possible          |
| Numerical precision       | Ranges acceptable, cite source |

### Content Guidelines

- Follow [Organization Style Guide]
- Adhere to [Ethics Policy]
- Disclose conflicts of interest
- Protect source confidentiality

### Review Process

All contributor content goes through:

1. **[Regional Lead] review** — Regional accuracy check
2. **Editorial review** — Style and quality
3. **Fact-check** — Verification of key claims
4. **Legal review** — Sensitive content only
