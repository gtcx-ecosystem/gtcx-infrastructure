# Security Reports

This directory contains generated security reports. Reports are created by the security scanning tools and should be reviewed regularly.

## Report Types

| Report | Generated By | Frequency |
|--------|--------------|-----------|
| `current-status.json` | `security-status.js` | On-demand |
| `audit-YYYY-MM-DD.json` | `pnpm audit` | CI/CD |
| `scan-YYYY-MM-DD.json` | Security scanners | Weekly |

## Usage

```bash
# Generate current status report
node infrastructure/security/scripts/security-status.js

# Run full audit
pnpm audit --json > infrastructure/security/reports/audit-$(date +%Y-%m-%d).json
```

## Retention

- Keep latest 10 reports
- Archive older reports to secure storage
- Reports may contain sensitive information
