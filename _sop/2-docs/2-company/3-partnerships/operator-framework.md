## Verification Tiers

### Listed (Free)

**Requirements**:

- Basic registration (name, contact, location)
- Operator type declaration
- [Jurisdiction] of operation

**Benefits**:

- Appear in [Registry Name] directory
- Basic profile page
- Claim your identity before others do

**Verification Level**: Self-reported, unverified

---

### Verified ([Price Range]/year)

**Requirements**:

- Documentation submitted and confirmed:
  - Business registration
  - Operating licenses
  - Tax registration
  - Identity verification (key personnel)
- Physical address confirmed
- Active operations confirmed

**Benefits**:

- "Verified" badge on profile
- Appear in verified-only searches
- Access to counterparty data
- Monthly digest of [domain] changes
- Priority in [buyer/partner] searches

**Verification Level**: Documentation reviewed, identity confirmed

---

### Certified ([Price Range]/year)

**Requirements**:

- All Verified requirements, plus:
- Physical inspection completed
- [Compliance Score] calculated
- Annual audit/reassessment
- Compliance documentation current

**Benefits**:

- "Certified" badge on profile
- [Compliance Score] displayed
- Featured in [Registry Name] reports
- Direct [buyer/partner] introductions
- Full [platform] access included
- [Partner] visibility

**Verification Level**: Field-verified, compliance scored

## Verification Process

### Listed (Immediate)

```
1. Online registration ([N] minutes)
2. Email verification
3. Basic profile creation
4. Immediately listed
```

### Verified (2-4 weeks)

```
1. Submit documentation package:
   - Business registration
   - Operating licenses
   - Tax certificates
   - ID verification

2. [Organization] review ([N]-[N] business days)

3. Clarification/additional docs (if needed)

4. Verification confirmed

5. Verified badge activated
```

### Certified (4-8 weeks)

```
1. Complete Verified process

2. Schedule field inspection

3. Inspector visit:
   - Physical presence confirmed
   - Operations observed
   - Documentation cross-checked
   - Stakeholder interviews

4. [Compliance Score] assessment calculated

5. Report and certification issued

6. Certified badge activated

7. Annual reassessment scheduled
```
