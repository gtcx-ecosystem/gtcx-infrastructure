## Tone & Voice

### We Sound Like

| ✅ Do                          | ❌ Don't                    |
| ------------------------------ | --------------------------- |
| Knowledgeable colleague        | Academic lecturer           |
| Trusted advisor                | Consultant (vague)          |
| Analyst who's done the work    | Press release (promotional) |
| Insider who respects your time | AI-generated (generic)      |

### Writing Principles

| Principle          | Application                                                        |
| ------------------ | ------------------------------------------------------------------ |
| **Authoritative**  | "[Authority] has..." not "It appears that..."                      |
| **Accessible**     | Complex ideas, clear language                                      |
| **Actionable**     | Always include "what this means for you"                           |
| **[Market]-first** | "[Country] announced" not "The [adjective] nation of [Country]..." |
| **Honest**         | Acknowledge uncertainty when it exists                             |
