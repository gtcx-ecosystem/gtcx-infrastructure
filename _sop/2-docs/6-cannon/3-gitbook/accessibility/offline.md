# Offline Access

[Organization Name] works without internet connectivity.

---

## How It Works

Our Progressive Web App (PWA) caches content for offline reading:

| Content Type               | Cache Duration |
| -------------------------- | -------------- |
| Breaking alerts            | 7 days         |
| Daily digests              | 7 days         |
| Country profiles           | 30 days        |
| [Index A]/[Index C] scores | 60 days        |
| Reports (downloaded)       | Until deleted  |

---

## Enabling Offline Mode

### Mobile (PWA)

1. Visit [organization-url] on mobile browser
2. Tap "Add to Home Screen"
3. Content automatically syncs when connected
4. Read cached content anytime

### Desktop

1. Visit [organization-url] in Chrome/Edge
2. Click install icon in address bar
3. Access from desktop with or without internet

---

## Sync Behavior

| Connectivity    | Behavior                              |
| --------------- | ------------------------------------- |
| **Online**      | Real-time updates, full functionality |
| **Offline**     | Read cached content, queue actions    |
| **Reconnected** | Sync queued actions, refresh cache    |

---

## Downloaded Reports

Download full reports for offline reading:

1. Open any [Platform C] report
2. Click **Download for Offline**
3. Report saved to device storage
4. Access anytime without internet
