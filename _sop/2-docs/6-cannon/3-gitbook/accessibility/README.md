# Global Accessibility

---

## Design Philosophy

> "Universal access requires designing for constraints, not capabilities."

We build for the hardest environments first. If it works in [low-connectivity / frontier markets] on 2G, it works everywhere.

---

## Accessibility Features

| Feature             | Description                                     |
| ------------------- | ----------------------------------------------- |
| **11 Languages**    | Tier 1: EN, FR, PT, AR, SW, HA + 5 regional     |
| **Offline-First**   | 7-day content cache, works without internet     |
| **Multi-Channel**   | Email, WhatsApp, SMS, USSD, Telegram, API       |
| **Low-Bandwidth**   | <10KB alerts, 70-85% compression                |
