## What belongs here

- UI component specifications
- Design system documentation
- UX flow diagrams and wireframes
- Interaction patterns and animation standards
- Design token definitions (colors, spacing, typography)
- Figma file references and design handoff notes

## What does NOT belong here

- Frontend implementation specs (→ `3-frontend/`)
- Accessibility standards (→ `4-engineering/5-compliance/accessibility/`)
- Brand identity (→ `2-company/1-brand-identity/`)
