- Cloud infrastructure topology and architecture
- Environment definitions (dev, staging, production)
- Deployment pipeline documentation
- Scaling configuration and disaster recovery procedures

## What does NOT belong here

- CI/CD pipeline scripts (→ `5-devops/3-ci-cd-pipelines/`)
- Operational runbooks (→ `5-devops/2-runbooks/`)
- Monitoring configuration (→ `5-devops/6-monitoring/`)
