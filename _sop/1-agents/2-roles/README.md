# Roles

SOPs and templates for defining contributor and team roles within an organization.

## Contents

| Path                                                         | Description                                                              |
| ------------------------------------------------------------ | ------------------------------------------------------------------------ |
| [contributors.md](contributors.md)                           | Contributor network roles, tiers, onboarding, and compensation framework |
| [role-identification-guide.md](role-identification-guide.md) | How to audit role coverage and identify missing roles                    |
| [examples/](examples/)                                       | Reference examples showing how to apply these templates                  |

## How to Use

1. Copy the relevant template file.
2. Replace all `[Placeholder]` values with organization-specific information.
3. Remove sections that do not apply.
4. Store completed role definitions in `docs/operations/organization/roles/`.
