| File                                             | Description                                                                             |

## What belongs here

- User story templates
- Filled story documents per feature
- Story-level acceptance criteria
- Story sizing and priority notes

## What does NOT belong here

- Epic-level breakdowns (→ `../3-features/`)
- Sprint plans (→ `../5-sprints/`)
- Backlog items (→ `../8-backlog/`)
