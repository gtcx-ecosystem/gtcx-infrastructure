# Features

Epic-level feature definitions with sprint scope, tasks, and UAT checklists.

## Contents

| File                                     | Description                                                                            |
| ---------------------------------------- | -------------------------------------------------------------------------------------- |
| [epic.md](epic.md)                       | Epic template — sprint goal, task breakdown by epic, UAT checklist, definition of done |
| [feature-backlog.md](feature-backlog.md) | Feature backlog structure — epic hierarchy, priority matrix, story point guide         |

## What belongs here

- Epic templates and filled epic documents
- Feature-level task breakdowns
- UAT checklists tied to specific epics
- Acceptance criteria per epic

## What does NOT belong here

- Phased roadmaps (→ `../2-phases/`)
- Individual user stories (→ `../4-stories/`)
- Sprint plans (→ `../5-sprints/`)
