# Protocol: Proposal Process

## Version

1.0

## Summary

Defines when and how design proposals and RFCs are required.

## When a Proposal is Required

- Architectural changes that affect multiple services
- New externally visible APIs
- Security or compliance-impacting changes
- Platform or infrastructure shifts

## Review Requirements

- At least one technical reviewer
- Stakeholder review if user impact is significant
