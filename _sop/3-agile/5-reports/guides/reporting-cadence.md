# Guide: Reporting Cadence

Recommended cadence and ownership for standard reporting.

## Cadence

| Report               | Cadence              | Owner            |
| -------------------- | -------------------- | ---------------- |
| Sprint update        | Bi-weekly            | Engineering lead |
| Sprint review        | End of sprint        | Engineering lead |
| Roadmap update       | Monthly              | Product lead     |
| Release notes        | Every release        | Release manager  |
| Production readiness | Before major release | Tech lead        |
