# 2 — Raw Transcript Files

> Unprocessed session transcripts. Store here before analysis.

---

## Filing Convention

Name files by date and topic:

`YYYY-MM-DD-[topic].md` or `YYYY-MM-DD-[topic].txt`

---

## Workflow

1. Save raw transcript here
2. Analyse in `3-transcript-analysis/` using the transcript analysis template
3. Extract insights to `4-session-insights/` and recaps to `5-session-recaps/`
