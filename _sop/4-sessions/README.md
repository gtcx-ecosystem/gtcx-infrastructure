# Sessions — gtcx-protocols

Session handoff documents and context recovery snapshots.

## What Belongs Here

- Session recap documents when ending a session mid-task
- Context snapshots before a long gap in work
- Handoff notes when handing work to another agent or team member

## Naming Convention

`session-{YYYY-MM-DD}-{topic}.md`

Example: `session-2026-03-06-pvp-escrow-refactor.md`

Use `_archive/legacy/legacy-SOPs/templates/session-docs/session-recap.md` as a starting point.
