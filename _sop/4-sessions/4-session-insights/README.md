## When to Use Each

| Template         | Trigger                                                              |
| ---------------- | -------------------------------------------------------------------- |
| **Flags**        | Something feels wrong, risky, or blocked                             |
| **Ideas**        | An interesting possibility surfaces — capture it now, evaluate later |
| **Innovation**   | A concept that could fundamentally change the value of the project   |
| **Observations** | A pattern is emerging across sessions, the codebase, or the team     |
