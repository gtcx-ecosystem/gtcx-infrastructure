This roadmap covers the **6-sprint horizon** from MOB-W1 closure through production hardening, GitOps maturity, and multi-region sovereign readiness.

## Universal Definition of Done

Every story is "done" only when **all** of the following hold:

1. **Code merged to `main`** with passing CI (all 17 validation gates)
2. **Kustomize build succeeds** for all affected overlays (staging, production, linkerd variants)
3. **Manifests applied** to at least one non-production environment and verified healthy
4. **Evidence committed:** release evidence manifest, SBOM, or test results in `infra/security/reports/`
5. **Documentation updated:** runbook, SLO definition, or architecture diagram if the change affects operations
6. **Rollback path documented:** how to revert the change in < 15 minutes
