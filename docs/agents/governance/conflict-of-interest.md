# Conflict of Interest Policy

> **Status:** Current
> **Date:** 2026-05-10
> **Owner:** GTCX Infrastructure

> Protecting [Organization Name]' integrity and credibility

---

## Purpose

[Organization Name]' value depends on trust. Our intelligence products, analysis, and recommendations must be free from bias created by personal, financial, or professional conflicts. This policy defines what conflicts exist, how to disclose them, and how to manage them.

---

## Scope

This policy applies to:

- All employees (full-time and part-time)
- All contributors (freelance and contract)
- All advisors and board members
- Any person producing content under [Organization Name] brand

---

## Types of Conflicts

### Financial Conflicts

| Conflict                  | Definition                               | Example                                              |
| ------------------------- | ---------------------------------------- | ---------------------------------------------------- |
| **Direct ownership**      | Owning securities in covered companies   | Owning shares in a [sector] company we cover         |
| **Indirect ownership**    | Family member ownership                  | Spouse works for covered refinery                    |
| **Trading on coverage**   | Trading based on unpublished information | Buying securities before publishing bullish analysis |
| **Payment from subjects** | Receiving money from covered entities    | Consulting fee from mining company                   |

### Professional Conflicts

| Conflict                 | Definition                            | Example                              |
| ------------------------ | ------------------------------------- | ------------------------------------ |
| **Prior employment**     | Recent work for covered entity        | Former employee of covered company   |
| **Future employment**    | Seeking job with covered entity       | Interviewing at company we cover     |
| **Board/advisory roles** | Serving covered entities              | Advisory board of mining association |
| **Speaking fees**        | Paid appearances for covered entities | Keynote at industry conference       |

### Personal Conflicts

| Conflict                   | Definition                                   | Example                                   |
| -------------------------- | -------------------------------------------- | ----------------------------------------- |
| **Family relationships**   | Relatives at covered entities                | Brother is CEO of covered company         |
| **Close friendships**      | Personal relationships affecting objectivity | Close friend is source                    |
| **Political affiliations** | Relationships with government officials      | Party member covering ruling party policy |

## Disclosure Requirements

### Mandatory Disclosure

All covered persons must disclose:

| Item                         | Frequency          | To Whom         |
| ---------------------------- | ------------------ | --------------- |
| Securities holdings (direct) | Annually + changes | Editor-in-Chief |
| Securities holdings (family) | Annually + changes | Editor-in-Chief |
| Outside income sources       | Annually           | Editor-in-Chief |
| Board/advisory positions     | Annually + changes | Editor-in-Chief |
| Prior employment (5 years)   | On hire            | HR              |
| Family at covered entities   | On hire + changes  | Editor-in-Chief |

### Disclosure Form

```yaml
disclosure:
  date: 2026-01-27
  period: Annual

  securities:
    - company: 'Example Mining Corp'
      ticker: 'EMC.JO'
      shares: 500
      relationship: 'Direct ownership'

  outside_positions:
    - organization: 'Mining Industry Association'
      role: 'Advisory board member'
      compensation: 'Unpaid'

  family_connections:
    - relative: 'Spouse'
      organization: '[Regulatory Agency]'
      role: 'Senior Analyst'

  prior_employment:
    - company: '[Company Name]'
      role: 'Communications Manager'
      dates: '2019-2022'
```

### Absolute Prohibitions

The following are **never permitted**:

| Activity                                   | Rationale                |
| ------------------------------------------ | ------------------------ |
| Trading on material non-public information | Securities law violation |
| Accepting payment from covered entities    | Compromises independence |
| Plagiarism                                 | Destroys credibility     |
| Fabricating sources                        | Destroys credibility     |
| Undisclosed conflicts in published work    | Misleads audience        |

### Restricted Activities

The following require **prior approval**:

| Activity                            | Approval By     |
| ----------------------------------- | --------------- |

| Outside writing/speaking for pay    | Editor-in-Chief |
| Consulting engagements              | CEO             |
| Board positions (any)               | CEO             |
| Political contributions >$500       | Editor-in-Chief |

## Recusal Requirements

### When to Recuse

You must recuse from coverage when:

| Situation                                   | Action                          |
| ------------------------------------------- | ------------------------------- |
| You own securities in the company           | Cannot cover that company       |
| Family member works there                   | Cannot cover that company       |
| You're seeking employment there             | Cannot cover until resolved     |
| You have personal relationship with subject | Cannot cover that person        |
| Prior employer (<2 years)                   | Cannot cover without disclosure |

### Recusal Process

1. **Identify conflict** — Recognize the situation
2. **Notify editor** — Inform immediately
3. **Document** — Record in conflict log
4. **Reassign** — Editor assigns to non-conflicted person
5. **Disclose if published** — If unavoidable, disclose in content

## Violations

### Reporting

Report suspected violations to:

- Editor-in-Chief (editorial matters)
- Legal counsel (serious matters)
- Board chair (if EIC involved)

### Consequences

| Severity     | Examples                               | Consequences                |
| ------------ | -------------------------------------- | --------------------------- |
| **Minor**    | Late disclosure, inadvertent omission  | Warning, training           |
| **Moderate** | Failure to recuse, repeated minor      | Suspension, reassignment    |
| **Serious**  | Undisclosed conflict in published work | Termination, correction     |
| **Severe**   | Trading on inside information          | Termination, legal referral |

### Appeals

Accused persons may appeal to:

1. Editor-in-Chief (for editor decisions)
2. CEO (for EIC decisions)
3. Board (for CEO decisions)
