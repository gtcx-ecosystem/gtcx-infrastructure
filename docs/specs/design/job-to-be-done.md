## Framework

A job statement follows the format:

> **When** [situation], **I want to** [motivation / goal], **so I can** [expected outcome].

Jobs are categorized as:

- **Functional** — practical tasks and outcomes
- **Emotional** — how users want to feel
- **Social** — how users want to be perceived

## Job Priority Matrix

| Job     | Importance to User | Current Satisfaction | Priority   |
| ------- | ------------------ | -------------------- | ---------- |
| [Job 1] | High               | Low                  | Build now  |
| [Job 2] | High               | Medium               | Improve    |
| [Job 3] | Medium             | Low                  | Build next |
| [Job 4] | Low                | High                 | Maintain   |

_Importance: how much the user cares about this outcome_
_Satisfaction: how well current solutions (including ours) address it_
_Opportunity = Importance + (Importance - Satisfaction)_
