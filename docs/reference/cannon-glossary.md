## S

**[Term S1]** — [Definition]

**SLI (Service Level Indicator)** — A specific, measurable metric used to assess service health (e.g., request success rate).

**SLO (Service Level Objective)** — A target value for an SLI (e.g., availability ≥ 99.9% over a 30-day window).

**SLA (Service Level Agreement)** — A formal commitment to customers about service performance, with defined consequences for breaches.
