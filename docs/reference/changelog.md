## Versioning Policy

| Increment         | When to Use                                |
| ----------------- | ------------------------------------------ |
| **Major (X.0.0)** | Breaking API changes, major product pivots |
| **Minor (0.X.0)** | New features, non-breaking changes         |
| **Patch (0.0.X)** | Bug fixes, security patches, documentation |
