## What Changed This Cycle

### Production Readiness: 8 -> 9

- Added explicit `strategy.rollingUpdate` with `maxUnavailable: 0` to all 10 deployments (zero-downtime deploys)
- Added `startupProbe` to 9 deployments that were missing them (api, crypto, anisa, intelligence-sdk, nats, prometheus, grafana, loki, jaeger)
- Hardened security contexts on intelligence services: added `readOnlyRootFilesystem`, `allowPrivilegeEscalation: false`, `capabilities.drop: ALL`, `/tmp` emptyDir volume
- Added full pod + container security context to Jaeger (was completely missing `runAsNonRoot`, `allowPrivilegeEscalation`, etc.)
- Added container security contexts (`allowPrivilegeEscalation: false`, `capabilities.drop: ALL`) to Prometheus, Grafana, Loki
- Remaining gap: load test results, alert calibration (both blocked on live traffic)

### Testability: 8 -> 9

- Added `backup.tftest.hcl`: 7 runs validating S3, KMS rotation, public access blocks, Lambda timeout, Glacier lifecycle
- Added `detective.tftest.hcl`: 8 runs validating CloudTrail log validation, GuardDuty, bucket security, 7-year retention
- Added `compliance.tftest.hcl`: 9 runs validating Config recorder, S3 security, all 7 compliance rules exist
- Added `alb.tftest.hcl`: 7 runs validating ACM, IRSA role, WAF scope/rules/rate-limiting, conditional ACM
- Coverage: 7/14 modules now have tftest files (was 3/14)
- Remaining gap: ecr, event-bus, kyc-documents, secrets, compliance-db modules still lack tests
