# GTM Documentation Overview

> **Status:** Current
> **Date:** 2026-05-10
> **Owner:** GTCX Infrastructure

This collection tracks the regulatory, sandbox, and commercialization evidence needed to move the infrastructure stack toward pilot and institutional readiness.

## Scope

- `plans/` holds execution plans and market-entry workplans.
- `regulatory/` holds regulator-facing and control-environment material.
- `sandbox-application/` holds the application evidence package assembled for central-bank or regulator review.

## What This Does Not Cover

- Product protocol semantics owned by `gtcx-protocols`
- Customer-facing SDK or API guidance owned by the service repos
- Internal sprint management owned by `docs/agile/`
